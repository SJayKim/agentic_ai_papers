# Evaluating Long-Context Reasoning in LLM-Based WebAgents

> **논문 정보**: Andy Chung, Yichi Zhang, Kaixiang Lin, Aditya Rawal, Qiaozi Gao, Joyce Chai (University of Michigan, Amazon)
> **arXiv**: 2024.12 (NeurIPS 2025 Workshop: LAW)
> **코드**: N/A

## Overview

| 항목 | 내용 |
|------|------|
| **Problem** | LLM 기반 WebAgent가 긴 상호작용 이력에서 추론하는 능력이 거의 탐구되지 않았다. 현실에서 사용자는 여러 세션에 걸쳐 에이전트와 상호작용하며, 이전 작업의 정보를 참조하는 후속 작업을 요청할 수 있다. 긴 컨텍스트에서 WebAgent의 추론 성능이 어떻게 변화하는지 체계적 평가가 부재하다. |
| **Motivation** | 사용자가 "어제 저장한 영화를 구매해줘"라고 요청하면, 에이전트는 긴 상호작용 이력에서 해당 영화를 찾아야 한다. 기존 벤치마크는 정적 QA나 짧은 컨텍스트에 집중하여, 실제 장기 사용 시나리오에서의 action-taking 에이전트 성능을 평가하지 못한다. |
| **Limitation** | (1) 56개 태스크만 사용하여 규모가 작다. (2) 라이브 인터넷 환경에서 테스트하여 웹사이트 변경으로 재현성이 제한될 수 있다. (3) 노이즈 태스크가 특정 시나리오로 제한되어 현실의 다양한 상호작용 패턴을 완전히 반영하지 못할 수 있다. (4) iRAG 개선이 modest하여 근본적 해결책이 되지 못함. |

## Method

이 논문은 WebAgent의 긴 컨텍스트 추론 능력을 평가하는 벤치마크와, 이를 개선하기 위한 implicit RAG 접근을 제안한다.

1. **벤치마크 설계**
   - 태스크를 순차 의존적 서브태스크 A₁, A₂로 분할
   - A₂는 **Under-specification**: A₁의 결과를 참조해야만 완료 가능 (예: A₁에서 찾은 호텔 근처 레스토랑 검색)
   - A₂는 **Unambiguous**: A₁에만 의존, 주입된 다른 태스크와 혼동 없음
   - A₁ 완료 후, A₁과 A₂ 사이에 **무관한 태스크 trajectory를 노이즈로 주입** (25K~150K 토큰)

2. **태스크 소싱**
   - WebCanvas 데이터셋에서 543개 태스크 중 Claude-3.5/3.7로 성공한 56개 선별
   - LLM으로 태스크를 A₁, A₂로 분할 후 수동 검증
   - 노이즈 태스크는 A₁과 직교하는 것만 선별

3. **평가 프로토콜**
   - Rule-based 핵심 단계(key steps) 검증 — LLM judge의 변동성 배제
   - 태스크당 평균 4.5개 핵심 단계, 모든 핵심 단계 성공 시 태스크 성공
   - 4개 모델 평가: Claude-3.7, GPT-4.1, Llama-4-17B, o4-mini

4. **Implicit RAG (iRAG)**
   - 에이전트의 매 단계에서 추가 RAG 스텝 삽입
   - 단순화된 지시로 현재 서브태스크·상태·상호작용 이력에 대한 요약 생성
   - 요약을 컨텍스트에 추가 → 복잡한 지시를 서브 지시로 분해하는 효과

---

## Key Contribution

1. **장기 컨텍스트 WebAgent 벤치마크**: 순차 의존 서브태스크 + 노이즈 주입으로 실제적인 장기 사용 시나리오를 시뮬레이션하는 최초의 벤치마크.
2. **극적 성능 하락 실증**: 컨텍스트 길이 증가에 따라 성공률이 40-50%에서 10% 미만으로 급락하는 현상을 4개 SOTA 모델에서 일관되게 관찰.
3. **실패 모드 분석**: 에이전트가 루프에 빠지거나(같은 행동 반복), 원래 태스크 목표를 잃어버리는 주요 실패 패턴 식별.
4. **iRAG 접근**: 서브 지시 분해를 통한 modest한 개선을 보여주지만, 근본적 한계가 지속됨을 확인.

## Experiment & Results

**모델**: Claude-3.7, GPT-4.1, Llama-4-17B Maverick, o4-mini

**컨텍스트 길이**: No noise (기준), 25K, 50K, 75K, 100K, 150K 토큰

**핵심 결과 (성공률)**:
- **No noise (기준)**: Claude-3.7 ~50%, GPT-4.1 ~45%, o4-mini ~40%, Llama-4 ~35%
- **노이즈 주입 시 급락**: 모든 모델이 25K에서 이미 ~30%로 하락, 150K에서 **10% 미만**
- o4-mini(추론 모델)가 상대적으로 가장 강건 — 추론 모델이 장문맥 추론에 유리

**실패 모드 분석**:
- **루프**: 에이전트가 동일한 비효과적 행동을 스텝 한도까지 반복
- **목표 상실**: 긴 컨텍스트에서 원래 태스크 목표를 잊어버림
- **태스크 효율 급감**: 대부분 스텝 한도에 도달

**iRAG 결과 (150K)**:
- Claude-3.7, GPT-4.1, o4-mini에서 baseline 대비 개선 관찰
- 하지만 개선 폭은 modest — 근본적 장문맥 추론 한계가 지속
- 서브 지시 분해가 일부 도움이 되지만 충분하지 않음
