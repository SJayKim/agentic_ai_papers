# S²R: Teaching LLMs to Self-verify and Self-correct via Reinforcement Learning

> **논문 정보**: Ruotian Ma et al.
> **arXiv**: ACL 2025
> **코드**: N/A

---

## Overview

| 항목 | 내용 |
|------|------|
| **Problem** | LLM의 자기 검증(self-verify)과 자기 교정(self-correct) 능력을 학습시키려면 대규모 데이터와 비용이 필요하다는 인식이 지배적이다. 기존 방법은 외부 교사 모델이나 대량 학습 데이터에 의존. |
| **Motivation** | 소량의 행동 초기화 데이터(3.1K)만으로도 결과 수준(outcome-level) + 과정 수준(process-level) RL을 결합하면 효과적인 자기 검증·교정을 학습할 수 있다면, 자원 효율적인 자기 교정이 가능. |
| **Limitation** | 저자 언급: 수학 도메인 한정. 독자 관점: 3.1K 초기화 데이터의 품질에 민감할 수 있음. |

---

## Method

1. **행동 초기화 (3.1K 샘플)**: 소량의 자기 검증·교정 시연 데이터로 모델 초기화
2. **Outcome-level RL**: 최종 답변의 정확도를 보상으로 사용
3. **Process-level RL**: 중간 추론 단계의 품질을 보상으로 사용
4. 두 수준의 RL을 결합하여 검증과 교정을 동시에 학습

---

## Key Contribution

1. **극소량 데이터로 자기 교정 달성**: 3.1K 샘플만으로 Qwen2.5-Math-7B를 51.0%→81.6%로 향상
2. **검증+교정의 동시 학습**: Outcome + Process RL의 결합

---

## Experiment & Results

- **모델**: Qwen2.5-Math-7B
- **벤치마크**: 수학 추론
- 정확도: 51.0% → **81.6%** (+30.6%p)
- 3.1K 초기화 샘플만 사용하여 자원 효율적
