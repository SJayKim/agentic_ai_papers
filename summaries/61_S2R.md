# S²R: Teaching LLMs to Self-verify and Self-correct via Reinforcement Learning

> **논문 정보**: Ruotian Ma, Xuanyu Zhang, Xingxuan Li, Yiwei Wang, Shafiq Joty, Lidong Bing
> **arXiv**: 2502.12853 / ACL 2025
> **코드**: https://github.com/RUCAIBox/S2R

---

## Overview

| 항목 | 내용 |
|------|------|
| **Problem** | LLM이 추론 중 스스로 오류를 검증하고 수정하려면 대규모 감독 데이터나 외부 교사 모델이 필요하다는 인식이 지배적이다. 기존 방법들은 GPT-4 수준의 교사로부터 증류하거나 수만 건의 annotation 데이터를 요구한다. |
| **Motivation** | 소량(3.1K)의 행동 초기화 데이터만으로도 결과 수준(outcome-level) RL과 과정 수준(process-level) RL을 결합하면, 외부 교사 없이 효과적인 자기 검증·교정 능력을 학습할 수 있다. 데이터 효율성과 자원 절약의 실용적 대안. |
| **Limitation** | (1) 수학(경쟁 수학) 도메인에 한정, 코딩·논리 추론 등 일반화 미검증. (2) 3.1K 초기화 데이터의 품질과 구성에 민감할 수 있음. (3) Process-level RL 보상 설계가 복잡하며 노이즈 유발 가능성. |

---

## Method

### 1단계: 행동 초기화 (Behavior Initialization)
- 자기 검증과 교정 시연을 포함한 3.1K 고품질 데이터로 기반 모델 SFT 초기화
- "검증 → 오류 감지 → 교정 시도" 행동 패턴 학습을 위한 최소 시드
- 의도적으로 소량 설계하여 RL 단계의 탐색 다양성 보존

### 2단계: Outcome-level RL
- 최종 답변의 정확도(정답 여부)를 보상 신호로 사용하는 RLVR 적용
- 모델이 다양한 자기 교정 전략을 탐색하도록 유도
- 보상: 정답 +1, 오답 0/-1의 이진 시그널

### 3단계: Process-level RL
- 중간 추론 단계(검증/교정 단계)의 품질을 보상으로 사용하는 세밀한 RL
- 검증 행동이 실제로 오류를 탐지했는지, 교정이 논리적으로 타당한지 평가
- 최종 보상만으로는 강화되기 어려운 중간 단계의 행동 패턴을 명시적으로 학습

### Outcome + Process RL 결합
- Outcome RL이 전체 방향성 제공, Process RL이 세부 추론 단계 정교화
- 기존 outcome-only RL 대비 "정답 우연 달성" 문제 완화

---

## Key Contribution

1. **극소량 데이터로 자기 교정**: 3.1K 샘플로 Qwen2.5-Math-7B 51.0% → **81.6%** (+30.6%p)
2. **이중 RL 학습 체계**: Outcome-level + Process-level RL을 결합한 최초의 체계적 프레임워크
3. **교사 모델 불필요**: GPT-4 등 외부 모델 없이 자체 RL만으로 수학 추론 자기 교정 달성
4. **데이터 효율성**: 기존 대규모 SFT 대비 수십 배 적은 데이터로 동등 이상의 성능

---

## Experiment & Results

- **모델**: Qwen2.5-Math-7B
- **벤치마크**: MATH-500, AMC, AIME 등 경쟁 수학 추론

**핵심 결과**:
- 전체 정확도: 51.0% → **81.6%** (+30.6%p)
- Process RL 추가 효과: Outcome-only RL 대비 +4~6%p (MATH-500 기준)
- 3.1K 초기화 샘플로 수만 건 SFT 방법과 동등 이상 성능

**Ablation**:
- 행동 초기화 없이 RL만 → 성능 저하 → 소량 초기화의 필요성 검증
- Outcome RL만 → Process RL 결합 대비 열등 → 이중 RL 효과 입증
- Process RL 단독보다 Outcome + Process 결합이 안정적 수렴
