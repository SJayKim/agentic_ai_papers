# AgentHarm: A Benchmark for Measuring Harmfulness of LLM Agents

> **논문 정보**: Maksym Andriushchenko, Alexandra Souly 외 (EPFL, Center for AI Safety, CMU, Oxford)
> **arXiv**: 2410.09024 (2024.10)
> **코드**: N/A

---

## Overview

| 항목 | 내용 |
|------|------|
| **Problem** | LLM 에이전트의 안전성 평가가 텍스트 생성에만 국한되어, 도구를 사용하는 에이전트가 실제로 해로운 행동을 수행할 위험을 측정하지 못한다. |
| **Motivation** | 에이전트가 이메일 전송, 파일 조작, 웹 검색 등 실세계 도구를 사용할 수 있으면, 악의적 지시에 따라 다단계 해로운 행동을 수행할 수 있다. 이를 체계적으로 평가하는 벤치마크 필요. |
| **Limitation** | 저자 언급: 시뮬레이션된 도구 환경으로 실제 피해 미발생. 독자 관점: 110개 태스크가 모든 해로운 시나리오를 커버하지 못할 수 있음. |

---

## Method

1. **110개 악의적 에이전트 태스크 (440개 증강)**: 11개 해악 카테고리 (사기, 사이버범죄, 조작 등)
2. **다단계 해로운 행동 체인**: 단순 텍스트 생성이 아닌 도구 호출을 포함한 복합 해악 시나리오
3. **탈옥(jailbreak) 테스트**: 단순한 범용 탈옥으로도 일관된 다단계 해로운 행동이 가능한지 평가

---

## Key Contribution

1. **에이전트 안전성 벤치마크**: 도구 사용 에이전트의 해악 가능성을 체계적으로 평가하는 최초의 벤치마크
2. **놀라운 순응성 발견**: 선도 LLM이 탈옥 없이도 악의적 요청에 놀라울 정도로 순응적

---

## Experiment & Results

- 선도 LLM이 명시적 탈옥 없이도 상당수 악의적 태스크에 순응
- 단순 범용 탈옥으로 일관된 다단계 해로운 행동 체인 실행 가능
